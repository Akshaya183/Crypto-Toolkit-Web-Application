# Crypto-Toolkit-Web-Application
A web-based cryptography toolkit that implements Vigenère Cipher, RC4 stream cipher, and Diffie-Hellman key exchange with an interactive and user-friendly interface.
## Overview
This project is a web-based cryptography toolkit developed as part of an academic assignment. It demonstrates the implementation of classical and modern cryptographic algorithms with an interactive user interface.
## Features
-  Vigenère Cipher (Encryption & Decryption)
-  RC4 Stream Cipher (with HEX output support)
-  Diffie-Hellman Key Exchange
-  Modern Timeline Dashboard UI
-  Input validation with alert messages
##  Technologies Used
- HTML
- CSS
- JavaScript
